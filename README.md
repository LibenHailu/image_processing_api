# Image Processing API

This is an image processing api built with node.js with typescript

## Setup

To run this project, install it locally using git:

```
$ cd image_processing_api
$ npm install
$ npm run start
```

## Usage

pass the image you want to resize,width and height

Example: http://localhost:3000/api/images?filename=encenadaport&height=200&width=200
